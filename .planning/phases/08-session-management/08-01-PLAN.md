<task id="08-01-03" title="Implement updateSessionTabs and process name polling">
<read_first>
- app/src/main/java/com/termux/app/TermuxActivity.java (after 08-01-02 changes)
- app/src/main/java/com/termux/app/terminal/TermuxTerminalSessionActivityClient.java (session list access, setCurrentSession)
- app/src/main/java/com/termux/app/TermuxService.java (getTermuxSessions, getTermuxSessionsSize, getIndexOfSession)
</read_first>

<action>
In `TermuxActivity.java`, add the following methods:

1. `private void updateSessionTabs()`:
   - Guard: `if (mSessionTabLayout == null || mTermuxService == null) return;`
   - Remove `mSessionTabLayout`'s `OnTabSelectedListener` temporarily (store in field `mTabSelectedListener`) to avoid recursive switchToSession calls during tab rebuild.
   - Call `mSessionTabLayout.removeAllTabs();`
   - Loop through `mTermuxService.getTermuxSessions()`, for each session:
     - Create tab via `mSessionTabLayout.newTab()`
     - Set text via `tab.setText(getSessionTabLabel(i, termuxSession))`
     - Add tab: `mSessionTabLayout.addTab(tab, false)` (false = don't select)
   - Select the tab matching the current session: find current session index via `mTermuxService.getIndexOfSession(mTerminalView.getCurrentSession())`, then `mSessionTabLayout.selectTab(mSessionTabLayout.getTabAt(currentIndex))`
   - Re-add `mTabSelectedListener`

2. `private String getSessionTabLabel(int index, TermuxSession termuxSession)`:
   - Get PID: `termuxSession.getExecutionCommand().mPid`
   - If PID > 0, try reading `/proc/PID/cmdline`:
     ```java
     try (FileInputStream fis = new FileInputStream("/proc/" + pid + "/cmdline")) {
         byte[] buffer = new byte[256];
         int read = fis.read(buffer);
         if (read > 0) {
             int end = 0;
             while (end < read && buffer[end] != 0) end++;
             String fullPath = new String(buffer, 0, end);
             int slash = fullPath.lastIndexOf('/');
             return slash >= 0 ? fullPath.substring(slash + 1) : fullPath;
         }
     } catch (IOException e) { /* fall through */ }
     ```
   - Fallback: `termuxSession.getTerminalSession().mSessionName` if non-null and non-empty
   - Final fallback: `"Session " + (index + 1)`

3. Process name polling `Runnable mProcessNamePoller`:
   ```java
   private final Runnable mProcessNamePoller = new Runnable() {
       @Override
       public void run() {
           updateTabLabelsOnly();
           mProcessNamePollHandler.postDelayed(this, 2000);
       }
   };
   ```

4. `private void updateTabLabelsOnly()`:
   - Guard: `if (mSessionTabLayout == null || mTermuxService == null) return;`
   - Loop tabs: for each tab index, get session, call `getSessionTabLabel()`, call `tab.setText(label)` only if changed.

5. Start polling in `onStart()`: `mProcessNamePollHandler.post(mProcessNamePoller);`
6. Stop polling in `onStop()`: `mProcessNamePollHandler.removeCallbacks(mProcessNamePoller);`

Add imports: `java.io.FileInputStream`, `java.io.IOException`.
</action>

<task id="08-01-04" title="Implement horizontal swipe gesture for session switching">
<read_first>

- terminal-view/src/main/java/com/termux/view/TerminalView.java (onTouchEvent, GestureAndScaleRecognizer)
</read_first>

<action>
In `TermuxActivity.java`:

1. Create a `GestureDetector` field `mSessionSwipeDetector` initialized in `setupSessionTabBar()`:
   ```java
   mSessionSwipeDetector = new GestureDetector(this, new GestureDetector.SimpleOnGestureListener() {
       private static final int SWIPE_MIN_DISTANCE = 120;
       private static final int SWIPE_THRESHOLD_VELOCITY = 200;

       @Override
       public boolean onFling(MotionEvent e1, MotionEvent e2, float velocityX, float velocityY) {
           if (e1 == null || e2 == null) return false;
           float deltaX = e2.getX() - e1.getX();
           float deltaY = e2.getY() - e1.getY();
           if (Math.abs(deltaX) > Math.abs(deltaY)
               && Math.abs(deltaX) > SWIPE_MIN_DISTANCE
               && Math.abs(velocityX) > SWIPE_THRESHOLD_VELOCITY) {
               if (deltaX < 0) {
                   mTermuxTerminalSessionActivityClient.switchToSession(true);
               } else {
                   mTermuxTerminalSessionActivityClient.switchToSession(false);
               }
               return true;
           }
           return false;
       }
   });
   ```

2. Add public method `getSessionSwipeDetector()` returning `mSessionSwipeDetector`.

3. In `TermuxTerminalViewClient.onTouchEvent(MotionEvent event)`, before returning, pass the event to the swipe detector:
   ```java
   GestureDetector swipeDetector = mActivity.getSessionSwipeDetector();
   if (swipeDetector != null) {
       swipeDetector.onTouchEvent(event);
   }
   ```

Add imports: `android.view.GestureDetector`, `android.view.MotionEvent` to TermuxActivity.
</action>
