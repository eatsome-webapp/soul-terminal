Extend the Pigeon bridge with closeSession, switchSession, renameSession (HostApi) and onSessionListChanged (FlutterApi) so SOUL chat can programmatically manage terminal sessions.
</objective>

2. Implement `closeSession(Long id)`:
   ```java
   @Override
   public void closeSession(@NonNull Long id) {
       int index = id.intValue();
       if (index < 0 || index >= mTermuxService.getTermuxSessionsSize()) {
           Logger.logWarn(LOG_TAG, "closeSession: invalid index " + index);
           return;
       }
       if (mTermuxService.getTermuxSessionsSize() <= 1) {
           Logger.logWarn(LOG_TAG, "closeSession: cannot close last session");
           return;
       }
       TermuxSession session = mTermuxService.getTermuxSession(index);
       if (session != null) {
           new Handler(Looper.getMainLooper()).post(() ->
               session.getTerminalSession().finishIfRunning()
           );
       }
   }
   ```

3. Implement `switchSession(Long id)`:
   ```java
   @Override
   public void switchSession(@NonNull Long id) {
       int index = id.intValue();
       if (mActivity != null) {
           new Handler(Looper.getMainLooper()).post(() ->
               mActivity.getTermuxTerminalSessionClient().switchToSession(index)
           );
       }
   }
   ```

4. Implement `renameSession(Long id, String name)`:
   ```java
   @Override
   public void renameSession(@NonNull Long id, @NonNull String name) {
       int index = id.intValue();
       TermuxSession session = mTermuxService.getTermuxSession(index);
       if (session != null) {
           new Handler(Looper.getMainLooper()).post(() -> {
               session.getTerminalSession().mSessionName = name;
               if (mActivity != null) {
                   mActivity.termuxSessionListNotifyUpdated();
               }
           });
       }
   }
   ```

5. Fix `getCurrentTerminalSession()` to use the actual current session instead of the last one:
   ```java
   private TerminalSession getCurrentTerminalSession() {
       if (mActivity != null && mActivity.getTerminalView() != null) {
           return mActivity.getTerminalView().getCurrentSession();
       }
       // Fallback to last session if activity not available
       if (mTermuxService.isTermuxSessionsEmpty()) return null;
       TermuxSession lastSession = mTermuxService.getLastTermuxSession();
       return lastSession != null ? lastSession.getTerminalSession() : null;
   }
   ```

6. Add imports: `android.os.Handler`, `android.os.Looper`, `com.termux.app.TermuxActivity`.

7. In `TermuxActivity.java`, in `onServiceConnected()` where `TerminalBridgeImpl` is created/registered, call `bridgeImpl.setActivity(this)` to pass the activity reference.

8. Verify `getTermuxTerminalSessionClient()` accessor exists on `TermuxActivity`. If not, add a public accessor:

   public TermuxTerminalSessionActivityClient getTermuxTerminalSessionClient() {
       return mTermuxTerminalSessionActivityClient;
   }
   ```
   This is needed by `switchSession()` implementation which calls `mActivity.getTermuxTerminalSessionClient().switchToSession(index)`.

// Notify Flutter of session list change
if (mSoulBridgeApi != null && mTermuxService != null) {
    List<TerminalBridgeApi.SessionInfo> sessions = new ArrayList<>();
    List<TermuxSession> termuxSessions = mTermuxService.getTermuxSessions();
    for (int i = 0; i < termuxSessions.size(); i++) {
        TermuxSession ts = termuxSessions.get(i);
        TerminalSession terminalSession = ts.getTerminalSession();
        String sessionName = terminalSession.mSessionName != null
            ? terminalSession.mSessionName
            : "Session " + (i + 1);
        sessions.add(new TerminalBridgeApi.SessionInfo.Builder()
            .setId((long) i)
            .setName(sessionName)
            .setIsRunning(terminalSession.isRunning())
            .build());
    }
    mSoulBridgeApi.onSessionListChanged(sessions, reply -> {});
}
```
