# Plan 03: Database layer — Drift schema + ObjectBox models kopiëren

<objective>
De volledige database laag (Drift SQLite met 25 tabellen + 20 DAOs) en alle data models (8 source files met Freezed/ObjectBox annotaties) kopiëren, objectbox-model.json meenemen, en build_runner draaien om .g.dart en .freezed.dart te genereren.
</objective>

<must_haves>
- Drift database schema (soul_database.dart + 25 .drift table files) gekopieerd
- Alle 20 DAOs gekopieerd
- Alle 8 model source files gekopieerd (NIET .g.dart of .freezed.dart)
- objectbox-model.json gekopieerd naar flutter_module/lib/
- build_runner genereert alle .g.dart en .freezed.dart files succesvol
- Database paden gebruiken getApplicationDocumentsDirectory() (geen hardcoded paden)
</must_haves>

<tasks>

<task id="03.1">
<title>Kopieer Drift database schema en DAOs</title>
<read_first>
- ~/soul-app/apps/soul/lib/services/database/soul_database.dart
- ~/soul-app/apps/soul/lib/services/database/tables/ (lijst bestanden)
- ~/soul-app/apps/soul/lib/services/database/daos/ (lijst bestanden)
</read_first>
<action>
Kopieer de database directory structuur:

```bash
mkdir -p flutter_module/lib/services/database/tables
mkdir -p flutter_module/lib/services/database/daos
```

Kopieer source files:
```bash
# Database definitie
cp ~/soul-app/apps/soul/lib/services/database/soul_database.dart flutter_module/lib/services/database/

# Alle 25 .drift table definities
cp ~/soul-app/apps/soul/lib/services/database/tables/*.drift flutter_module/lib/services/database/tables/

# Alle 20 DAO source files (NIET .g.dart)
for f in ~/soul-app/apps/soul/lib/services/database/daos/*.dart; do
  [[ "$f" != *.g.dart ]] && cp "$f" flutter_module/lib/services/database/daos/
done
```

NOOIT soul_database.g.dart of *.g.dart files uit daos/ kopiëren — deze worden door build_runner gegenereerd.

Controleer of soul_database.dart `getApplicationDocumentsDirectory()` gebruikt voor het database pad (MERG-05 vereiste).
</action>
<acceptance_criteria>
- `flutter_module/lib/services/database/soul_database.dart` bestaat
- 25 .drift files in `flutter_module/lib/services/database/tables/`
- Geen .g.dart files in `flutter_module/lib/services/database/` (alleen source)
- `grep "getApplicationDocumentsDirectory" flutter_module/lib/services/database/soul_database.dart` vindt een match
</acceptance_criteria>
</task>

<task id="03.5">

<read_first>
- flutter_module/lib/services/database/soul_database.dart
- flutter_module/lib/objectbox-model.json
</read_first>
<action>
Draai build_runner om alle generated files aan te maken:

```bash

dart run build_runner build --delete-conflicting-outputs
```

Dit genereert:
- `soul_database.g.dart` (Drift)
- `objectbox.g.dart` (ObjectBox — in lib/ root)
- `*.g.dart` voor DAOs
- `*.freezed.dart` en `*.g.dart` voor models

build_runner zal waarschijnlijk FALEN omdat services/ nog niet gekopieerd is en providers.dart naar die files verwijst. Dit is VERWACHT. De build_runner fouten zijn informatief — ze tonen welke files nog ontbreken.

Als build_runner deels draait (Drift en ObjectBox generators apart van Freezed): verifieer dat de database generated files correct zijn.
