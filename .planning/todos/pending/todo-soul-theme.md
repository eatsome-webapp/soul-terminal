De huidige terminal heeft standaard Termux kleuren. SOUL heeft een eigen kleurpalet
(#0F0F23 achtergrond, #6C63FF accent, #00D9FF highlights).

## Aanpak
- `colors.properties` aanmaken in app assets met SOUL kleuren
- Syntax highlight kleuren optimaliseren voor leesbaarheid van code output
- Cursor kleur: #6C63FF (SOUL paars)
- Selectie highlight: #6C63FF met transparantie

## Kleuren
- Achtergrond: #0F0F23
- Tekst: #E0E0E0
- Accent/cursor: #6C63FF
- Groen (succes): #00D9FF
- Rood (error): #FF6B6B
