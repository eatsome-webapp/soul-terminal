# Plan 02: Core layer + providers kopiëren en build_runner draaien

<objective>
De volledige core/ directory van de SOUL app kopiëren naar flutter_module, inclusief providers.dart (60+ Riverpod providers), router, theme, sentry config, en utilities — en verifiëren dat build_runner slaagt.
</objective>

<must_haves>
- Alle 6 core source files gekopieerd naar flutter_module/lib/core/
- providers.dart bevat alle 60+ provider definities ongewijzigd (behalve evt. import paden)
- Geen `package:soul_app/` imports — alle imports zijn relatief of `package:flutter_module/`
- build_runner slaagt (voor zover core files codegen bevatten)
- CI build groen na push
</must_haves>

- ~/soul-app/apps/soul/lib/core/theme/soul_theme.dart

Kopieer ook utils/ (check hoeveel bestanden er in zitten):

NOOIT .g.dart of .freezed.dart files kopiëren.
