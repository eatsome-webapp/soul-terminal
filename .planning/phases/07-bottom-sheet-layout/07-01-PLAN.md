# Plan 07-01: CoordinatorLayout refactor + BottomSheet with drag handle

<objective>
Replace the TermuxActivityRootView (LinearLayout) root with CoordinatorLayout. Make FlutterFragment the fullscreen background. Wrap the terminal hierarchy in a BottomSheetBehavior child with a dedicated drag handle. Configure 4 sheet states: hidden, collapsed (48dp peek), half-expanded (40%), expanded (fullscreen). Remove the old toggleFlutterView() mechanism and replace with sheet state management.
</objective>

**15. Add new method `setupBottomSheet()`:**
```java
private void setupBottomSheet() {
    View sheetContainer = findViewById(R.id.terminal_sheet_container);
    mBottomSheetBehavior = BottomSheetBehavior.from(sheetContainer);
    mBottomSheetBehavior.setState(BottomSheetBehavior.STATE_COLLAPSED);

    mBottomSheetBehavior.addBottomSheetCallback(new BottomSheetBehavior.BottomSheetCallback() {
        @Override
        public void onStateChanged(@NonNull View bottomSheet, int newState) {
            if (newState == BottomSheetBehavior.STATE_EXPANDED && mTerminalView != null) {
                mTerminalView.requestFocus();
            }
            Logger.logDebug(LOG_TAG, "Sheet state changed to: " + newState);
        }

        @Override
        public void onSlide(@NonNull View bottomSheet, float slideOffset) {
            // No-op for now
        }
    });

    // IME insets handling on sheet container
    ViewCompat.setOnApplyWindowInsetsListener(sheetContainer, (view, insets) -> {
        int imeHeight = insets.getInsets(WindowInsetsCompat.Type.ime()).bottom;
        int navBarHeight = insets.getInsets(WindowInsetsCompat.Type.navigationBars()).bottom;
        view.setPadding(0, 0, 0, Math.max(imeHeight, navBarHeight));
        return insets;
    });

    // Back button handling
    getOnBackPressedDispatcher().addCallback(this, new OnBackPressedCallback(true) {
        @SuppressLint("RtlHardcoded")
        @Override
        public void handleOnBackPressed() {
            if (getDrawer().isDrawerOpen(Gravity.LEFT)) {
                getDrawer().closeDrawers();
                return;
            }
            int state = mBottomSheetBehavior.getState();
            if (state == BottomSheetBehavior.STATE_EXPANDED
                    || state == BottomSheetBehavior.STATE_HALF_EXPANDED) {
                mBottomSheetBehavior.setState(BottomSheetBehavior.STATE_COLLAPSED);
            } else {
                finishActivityIfNotFinishing();
            }
        }
    });
}
```
