# Plan 05-01: Terminal defaults en SOUL branding

<objective>
Verhoog scrollback buffer naar 20.000 regels, stel Claude Code-geoptimaliseerde extra keys in, pas terminal kleuren aan naar SOUL thema, en trek SOUL branding door naar drawer en app chrome.
</objective>

</read_first>

<task id="05-01-02">
<title>Extra keys layout optimaliseren voor Claude Code</title>
<read_first>
- termux-shared/src/main/java/com/termux/shared/termux/settings/properties/TermuxPropertyConstants.java
- .planning/phases/05-terminal-quick-wins/05-RESEARCH.md
</read_first>
<action>
In `termux-shared/src/main/java/com/termux/shared/termux/settings/properties/TermuxPropertyConstants.java`, regel 329:

Wijzig:
```java
public static final String DEFAULT_IVALUE_EXTRA_KEYS = "[['ESC','/',{key: '-', popup: '|'},'HOME','UP','END','PGUP'], ['TAB','CTRL','ALT','LEFT','DOWN','RIGHT','PGDN']]"; // Double row
```

Naar:
```java
public static final String DEFAULT_IVALUE_EXTRA_KEYS = "[['ESC','TAB',{macro:'CTRL C',display:'^C'},{macro:'CTRL D',display:'^D'},{macro:'CTRL Z',display:'^Z'},{macro:'CTRL L',display:'^L'},'UP'],['HOME','END','LEFT','DOWN','RIGHT','CTRL','ALT']]"; // Claude Code optimized
```

Rij 1: ESC, TAB, ^C, ^D, ^Z, ^L, UP (Claude Code shortcuts)
Rij 2: HOME, END, LEFT, DOWN, RIGHT, CTRL, ALT (navigatie)
</action>
<acceptance_criteria>
- `grep "CTRL C" termux-shared/src/main/java/com/termux/shared/termux/settings/properties/TermuxPropertyConstants.java` vindt exact 1 match
- `grep "CTRL D" termux-shared/src/main/java/com/termux/shared/termux/settings/properties/TermuxPropertyConstants.java` vindt exact 1 match
- `grep "Claude Code optimized" termux-shared/src/main/java/com/termux/shared/termux/settings/properties/TermuxPropertyConstants.java` vindt exact 1 match
- De string bevat `macro` keyword (grep bevestigt macro syntax)
</acceptance_criteria>
</task>

<task id="05-01-03">
<title>SOUL kleurthema als terminal default instellen</title>
<read_first>
- terminal-emulator/src/main/java/com/termux/terminal/TerminalColorScheme.java
</read_first>
<action>
In `terminal-emulator/src/main/java/com/termux/terminal/TerminalColorScheme.java`, regel 60:



</acceptance_criteria>
</task>
