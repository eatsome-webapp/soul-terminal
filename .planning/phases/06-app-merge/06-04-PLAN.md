# Plan 04: Services layer kopiëren — alle 112 service files

<objective>
Alle 112 service source files kopiëren naar flutter_module, georganiseerd per subdirectory (ai, agentic, chat, demo, memory, monitoring, platform, success, vessels), met correcte foreground service configuratie (MERG-06) en API key service (MERG-09).
</objective>

<must_haves>
- Alle 112 service source files gekopieerd (NIET .g.dart of .freezed.dart)
- ForegroundServiceManager gebruikt channel ID `soul_foreground` en notification ID 256 (geen conflict met Termux IDs 1-10)
- ApiKeyService gebruikt FlutterSecureStorage (Android Keystore scoped aan com.soul.terminal)
- Geen `package:soul_app/` imports in enige service file
- Alle service subdirectories aanwezig: ai, agentic, auth (al gekopieerd), chat, database (al gekopieerd), demo, memory, monitoring, platform, success, vessels
</must_haves>
