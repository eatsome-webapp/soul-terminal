# Plan 01: Pubspec.yaml merge — alle dependencies toevoegen

<objective>
Alle 41 runtime en 9 dev dependencies uit de SOUL app toevoegen aan flutter_module/pubspec.yaml, inclusief SDK constraint update, assets sectie, en module configuratie behouden.
</objective>

<must_haves>
- flutter_module/pubspec.yaml bevat alle 41 SOUL runtime dependencies + bestaande pigeon dep
- flutter_module/pubspec.yaml bevat alle 9 dev dependencies (build_runner, drift_dev, freezed, json_serializable, riverpod_generator, envied_generator, flutter_lints, objectbox_generator)
- SDK constraint is `^3.11.1`
- `module:` sectie behouden met androidPackage: com.soul.terminal.flutter_module
- Assets sectie bevat `assets/config/default_intervention_config.yaml`
- `flutter pub get` slaagt zonder version conflicts
</must_haves>
