<action>
In `flutter_module/lib/main.dart`, add a crash recovery check inside `_initAndRun()`.

1. Add import at top of file:
```dart
import 'services/profile_pack/profile_pack_service.dart';
```

2. Insert the following block AFTER the foreground service start section (after the `_logger.i('Foreground service started');` catch block) and BEFORE the "Check if setup wizard has been completed" section:

```dart
// Check for interrupted profile pack installation (crash recovery — PROF-07)
try {
  final packService = ProfilePackService();
  final interrupted = await packService.getInterruptedInstallation();
  if (interrupted != null) {
    _logger.w('Interrupted profile pack installation detected: $interrupted');
    final settingsDao = container.read(settingsDaoProvider);
    await settingsDao.setString(SettingsKeys.interruptedProfileInstall, interrupted);
  }
} catch (error) {
  _logger.e('Crash recovery check failed: $error');
}
```

This stores the interrupted profile ID (e.g., "claude-code") in settings. The setup wizard or a future recovery dialog can read this key and offer retry/cleanup. The check is non-fatal — if it fails, the app continues normally.
</action>

Add a method to `ProfilePackService` that cleans up a failed installation by deleting the `.soul-profile-installing` marker file. This is needed for the "cleanup" option in the recovery dialog.

Add the following method after `getInterruptedInstallation()`:

```dart
/// Clean up an interrupted installation by removing the progress marker.
/// Call this when the user chooses "cleanup" instead of "retry".
Future<void> cleanupInterruptedInstallation() async {
  try {
    const markerPath = '/data/data/com.soul.terminal/files/usr/.soul-profile-installing';
    final file = File(markerPath);
    if (file.existsSync()) {
      file.deleteSync();
      _logger.i('Cleaned up interrupted installation marker');
    }
  } catch (e) {
    _logger.e('Failed to clean up installation marker: $e');
  }
}
```

This method uses direct File I/O rather than Pigeon because it's a simple file delete at a known path.
