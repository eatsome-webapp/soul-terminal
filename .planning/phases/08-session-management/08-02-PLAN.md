# Plan 08-02: Session actions (rename, close) and drawer cleanup

<objective>
Add long-press context menu on session tabs for rename and close actions. Clean up the drawer by removing the session list ListView and new session button, keeping only settings access.
</objective>

<tasks>

<task id="08-02-01" title="Add long-press context menu on tabs">
<read_first>
- app/src/main/java/com/termux/app/TermuxActivity.java (setupSessionTabBar, updateSessionTabs from Plan 01)
- app/src/main/java/com/termux/app/terminal/TermuxTerminalSessionActivityClient.java (renameSession, removeFinishedSession)
- app/src/main/java/com/termux/app/TermuxService.java (getTermuxSession, getTermuxSessionsSize, removeTermuxSession)
</read_first>

<action>
In `TermuxActivity.java`:

1. In `updateSessionTabs()`, after adding each tab, set a long-click listener on the tab view:
   ```java
   TabLayout.Tab tab = mSessionTabLayout.newTab();
   // ... setText ...
   mSessionTabLayout.addTab(tab, false);
   // Set long click on the tab view
   View tabView = ((ViewGroup) mSessionTabLayout.getChildAt(0)).getChildAt(i);
   final int sessionIndex = i;
   tabView.setOnLongClickListener(v -> {
       showSessionContextMenu(v, sessionIndex);
       return true;
   });
   ```

2. Create method `private void showSessionContextMenu(View anchor, int sessionIndex)`:
   ```java
   PopupMenu popup = new PopupMenu(this, anchor);
   popup.getMenu().add(0, 1, 0, "Hernoemen");
   popup.getMenu().add(0, 2, 0, "Sluiten");
   popup.setOnMenuItemClickListener(item -> {
       switch (item.getItemId()) {
           case 1: // Rename
               showRenameSessionDialog(sessionIndex);
               return true;
           case 2: // Close
               closeSessionAtIndex(sessionIndex);
               return true;
       }
       return false;
   });
   popup.show();
   ```

3. Create method `private void showRenameSessionDialog(int sessionIndex)`:
   - Get `TermuxSession` via `mTermuxService.getTermuxSession(sessionIndex)`
   - Create `android.app.AlertDialog.Builder` with an `EditText` input
   - Pre-fill with current session name
   - On positive button: set `terminalSession.mSessionName = newName`, then call `termuxSessionListNotifyUpdated()`

4. Create method `private void closeSessionAtIndex(int sessionIndex)`:
   - Get `TermuxSession` via `mTermuxService.getTermuxSession(sessionIndex)`
   - If `mTermuxService.getTermuxSessionsSize() <= 1`: show toast "Kan laatste sessie niet sluiten" and return
   - Call `termuxSession.getTerminalSession().finishIfRunning()` — the `onSessionFinished` callback in `TermuxTerminalSessionActivityClient` handles removal and tab update

Add imports: `android.widget.PopupMenu`, `android.widget.EditText`, `android.app.AlertDialog`, `android.view.ViewGroup`.
</action>
