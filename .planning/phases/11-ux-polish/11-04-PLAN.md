# Plan 11-04: Landscape layout & accessibility

<objective>
Create a landscape layout with 60/40 side-by-side split (SOUL chat left, terminal right) and add content descriptions plus TalkBack announcements to all interactive elements.
</objective>

<must_haves>
- `res/layout-land/activity_termux.xml` with horizontal LinearLayout: flutter_container 60%, terminal 40%
- No BottomSheetBehavior in landscape — terminal is always visible as side panel
- `onConfigurationChanged()` re-inflates layout and rebinds all view references
- `TerminalView.updateSize()` called after orientation change
- FlutterFragment re-attached after layout re-inflation
- Content descriptions on: sheet_drag_handle, terminal_view, toggle_keyboard_button, toggle_soul_button, new_session_tab_button (existing but update to Dutch), settings_tab_button (existing but update)
- Tab content descriptions set programmatically with session name
- TalkBack announcements via `announceForAccessibility()` on sheet state changes and session switches
- Accessibility strings in `res/values/strings.xml`
- `accessibilityLiveRegion="polite"` on terminal_view
- Blur/fling logic guarded with landscape check (no BottomSheetBehavior in landscape)
</must_haves>

2. Add a private `mIsLandscape` boolean field near the other fields (around r168):

```java
private boolean mIsLandscape = false;
```

3. Add `onConfigurationChanged()` method (after `onDestroy()` or `onStop()`):

```java
@Override
public void onConfigurationChanged(@NonNull Configuration newConfig) {
    super.onConfigurationChanged(newConfig);
    boolean wasLandscape = mIsLandscape;
    mIsLandscape = (newConfig.orientation == Configuration.ORIENTATION_LANDSCAPE);

    if (wasLandscape == mIsLandscape) return; // no orientation change

    // Re-inflate layout — Android picks layout-land/ for landscape automatically
    setContentView(R.layout.activity_termux);

    // Rebind all view references
    rebindViewsAfterLayoutChange();

    if (!mIsLandscape) {
        setupBottomSheet();
    }
    setupSessionTabBar();

    // Re-attach FlutterFragment
    FlutterFragment flutterFragment = (FlutterFragment) getSupportFragmentManager()
        .findFragmentByTag("flutter_fragment");
    if (flutterFragment == null) {
        flutterFragment = FlutterFragment.withCachedEngine("soul_engine").build();
        getSupportFragmentManager()
            .beginTransaction()
            .replace(R.id.flutter_container, flutterFragment, "flutter_fragment")
            .commit();
    }

    if (mTerminalView != null) {
        mTerminalView.updateSize();
    }

    Logger.logDebug(LOG_TAG, "Layout changed to " + (mIsLandscape ? "landscape" : "portrait"));
}

private void rebindViewsAfterLayoutChange() {
    mTerminalView = findViewById(R.id.terminal_view);
    if (mTerminalView != null && mTermuxTerminalViewClient != null) {
        mTerminalView.setTerminalViewClient(mTermuxTerminalViewClient);
    }
    if (mTerminalView != null && mTermuxTerminalSessionActivityClient != null) {
        TerminalSession currentSession = getCurrentSession();
        if (currentSession != null) {
            mTerminalView.attachSession(currentSession);
        }
    }
    // Re-bind terminal view touch listener for session swipe
    if (mTerminalView != null) {
        mTerminalView.setOnTouchListener((v, event) -> {
            if (mTermuxTerminalViewClient != null) {
                mTermuxTerminalViewClient.onTouchEvent(event);
            }
            return false;
        });
    }
}
```

```java
}
```
